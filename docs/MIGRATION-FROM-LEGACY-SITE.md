# Migration der alten Contracomology-Seite

Quelle: contracomology.org (Legacy-Version)

## Ziel

Die bestehende Seite wird nicht 1:1 kopiert.

Ihre Inhalte werden in vier Ebenen zerlegt:

1. Startseite
2. Kursmodule
3. KG-Begriffe
4. Dokumente

## Startseite

Beibehalten:

- Wo Bach den Urknall trifft
- Einführung in die Kontrakomologie
- kurze Orientierung für Erstbesucher

## Kursmodule

Aus den bisherigen Inhalten entstehen Lernmodule:

- Einführung
- Simultane Polyphonie
- Komplexe Einfachheit
- Plastische Zeit
- Kreative Unvollendung
- Kontrapunktisches Bewusstsein
